Программа hardload.exe разработана для копирования файлов из одной папки в другую. 
Программа запускается на персональном компьютере, имеющим операционную систему Windows.
Чтобы запустить программу, необходимо открыть файл hardload.exe
В данной программе есть несколько функций: 
1. Найти и добавить файл в список;
2. Выбор папки для копирования;
3. Копирование файла;
4. Есть возможность остановить копирование и убрать файлы из списка добавленных;
Также на экране программы присутствует список файлов, которые были добавлены для копирования.
Чтобы скопировать файл из одной папки в другую, необходимо выполнить следующие действия:
1. Нажать на кнопку "Найти и добавить файл в список" (В списке файлов появляется выбранный нами файл с указанием пути к нему);
2. Выделить в списке появившийся файл, нажав на "чекбокс";
3. Нажать на кнопку "Выбрать папку, куда писать файлы";
4. Выбрать нужную папку для копирования;
5. Нажать на кнопку "Начать грузить", начнется процедура копирования.

В приложении так же есть возможность удалить файл из списка файлов, 
выделив нужный файл левой кнопкой мыши и нажав на кнопку "Удалить выделенные файлы из списка".

Также есть возможность копирования сразу нескольких файлов, находящихся в списке. 
Достаточно поставить галочки на "чекбоксы" возле необходимых файлов и нажать на кнопку "Начать грузить".
Приложение может копировать файлы крупных размеров (1+ гб.).
Присутствует диалоговое окно, в котором указываются подсказки, что означает кнопка, на которую наведен курсор мыши.

ОГРАНИЧЕНИЯ:

-Приложение не может копировать файлы, которые не выделены в списке "чекбоксом", а также не может удалить сразу несколько файлов из списка, 
каждый файл необходимо выделять отдельно и нажать "Удалить выделенные файлы из списка".
-Приложение не может копировать файлы в папку, в которой находится нужный Вам файл для копирования, только лишь в другую папку.
-Нельзя добавить сразу несколько файлов в список, только каждый отдельно.
-Также, приложение не может копировать файлы с названием из специальных символов.
